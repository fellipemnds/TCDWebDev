Exemplo

Defina o parâmetro "autoplay" como "true", para que o som comece a tocar assim que a página carregar:

html

param

  1. <object data="horse.wav">
  2. <param name="autoplay" value="true">
  3. </object>

Definição e Uso

A tag <param> é usada para definir parâmetros para um elemento <object>

Atributos

Atributo Valor Descrição
name name Especifica o nome de um parâmetro
value value Especifica o valor do parâmetro
