Exemplo

Um vídeo com legendas em dois idiomas:

html

track

  1. <video width="320" height="240" controls>
  2. <source src="forrest_gump.mp4" type="video/mp4">
  3. <source src="forrest_gump.ogg" type="video/ogg">
  4. <track src="fgsubtitles_en.vtt" kind="subtitles" srclang="en" label="English">
  5. <track src="fgsubtitles_no.vtt" kind="subtitles" srclang="no" label="Norwegian">
  6. </video>

Definição e Uso

A tag <track> especifica faixas de texto para elementos <audio> ou <video> . Este elemento é usado para especificar legendas, arquivos de legendas ou outros arquivos contendo texto, que devem ser visíveis durante a reprodução da mídia. As trilhas são formatadas no formato WebVTT (arquivos .vtt).

Atributos

Atributo Valor Descrição
default default Especifica que a faixa deve ser habilitada se as preferências do usuário não indicarem que outra faixa seria mais apropriada
kind captions
chapters
descriptions
metadata
subtitles		 Especifica o tipo de faixa de texto
label text Especifica o título da faixa de texto
src URL Obrigatório. Especifica o URL do arquivo de trilha
srclang language_code Especifica o idioma dos dados de texto da trilha (obrigatório se kind = "subtitles")
