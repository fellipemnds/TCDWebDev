Tag HTML <output>

Exemplo

Faça um cálculo e mostre o resultado em um elemento <output>:

<p><form oninput="x.value=parseInt(a.value)+parseInt(b.value)">
<input type="range" id="a" value="50">
+<input type="number" id="b" value="25">
=<output name="x" for="a b"></output>
</form>

Definição e Uso

A tag <output> é usada para representar o resultado de um cálculo (como um executado por um script).

Atributos

Atributo Valor Descrição
for element_id Specifies the relationship between the result of the calculation, and the elements used in the calculation
form form_id Specifies which form the output element belongs to
name name Specifies a name for the output element
