Definição e Uso

A tag <td> define uma célula de dados padrão em uma tabela HTML.
Uma tabela HTML possui dois tipos de células:

  • Células de cabeçalho - contém informações de cabeçalho (criadas com o elemento <th> )
  • Células de dados - contém dados (criados com o <td> elemento)
O texto nos <td> elementos é regular e alinhado à esquerda por padrão.
O texto nos elementos <th> está em negrito e centralizado por padrão.

Atributos

Atributo Valor Descrição
colspan number Especifica o número de colunas que uma célula deve abranger
headers header_id Especifica uma ou mais células de cabeçalho às quais uma célula está relacionada
rowspan number Define o número de linhas que uma célula deve abranger
