Tag HTML <figure>

Exemplo

Use um elemento <figure> para marcar uma foto em um documento e um elemento <figcaption> para definir uma legenda para a foto:

<figure>
<img src="pic_trulli.jpg" alt="Trulli" style="width:100%">
<figcaption>Fig.1 - Trulli, Puglia, Italy.</figcaption>
</figure>

Definição e Uso

A <figure> tag especifica conteúdo independente, como ilustrações, diagramas, fotos, listagens de código, etc. Embora o conteúdo do <figure> elemento esteja relacionado ao fluxo principal, sua posição é independente do fluxo principal e, se removido, não deve afetar o fluxo do documento.
