Exemplo

Use um elemento <figure> para marcar uma foto em um documento e um elemento <figcaption> para definir uma legenda para a foto:

html

figure

  1. <figure>
  2. <img src="pic_trulli.jpg" alt="Trulli" style="width:100%">
  3. <figcaption>Fig.1 - Trulli, Puglia, Italy.</figcaption>
  4. </figure>

Definição e Uso

A tag <figure> especifica conteúdo independente, como ilustrações, diagramas, fotos, listagens de código, etc. Embora o conteúdo do <figure> elemento esteja relacionado ao fluxo principal, sua posição é independente do fluxo principal e, se removido, não deve afetar o fluxo do documento.
