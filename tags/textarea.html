Definição e Uso

A tag <textarea> define um controle de entrada de texto de várias linhas. O <textarea>elemento é frequentemente usado em um formulário, para coletar entradas do usuário, como comentários ou análises. Uma área de texto pode conter um número ilimitado de caracteres e o texto é renderizado em uma fonte de largura fixa (geralmente Courier). O tamanho de uma área de texto é especificado pelos atributos <cols>e <rows>(ou com CSS). O nameatributo é necessário para fazer referência aos dados do formulário após o formulário ser enviado (se você omitir o nameatributo, nenhum dado da área de texto será enviado). O idatributo é necessário para associar a área de texto a um rótulo.

Atributos

Atributo Valor Descrição
autofocus autofocus Especifica que uma área de texto deve receber foco automaticamente quando a página é carregada
cols number Especifica a largura visível de uma área de texto
dirname textareaname.dir Especifica que a direção do texto da área de texto será enviada
disabled disabled Especifica que uma área de texto deve ser desativada
form form_id Specifies which form the text area belongs to
maxlength number Especifica a qual formulário a área de texto pertence
name text Especifica um nome para uma área de texto
placeholder text Especifica uma pequena dica que descreve o valor esperado de uma área de texto
readonly readonly Especifica que uma área de texto deve ser somente leitura
required required Especifica que uma área de texto é necessária/deve ser preenchida
rows number Especifica o número visível de linhas em uma área de texto
wrap hard
soft		 Especifica como o texto em uma área de texto deve ser quebrado quando submetido em um formulário
