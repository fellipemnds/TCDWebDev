Tag HTML <data>

Exemplo

O exemplo a seguir exibe nomes de produtos, mas também associa cada nome a um número de produto:

<ul>
<li><data value="21053">Cherry Tomato</data></li>
<li><data value="21054">Beef Tomato</data></li>
<li><data value="21055">Snack Tomato</data></li>
</ul>

Definição e Uso

A tag <data> é usada para adicionar uma tradução legível por máquina de um determinado conteúdo.Este elemento fornece um valor legível por máquina para processadores de dados e um valor legível por humanos para renderização em um navegador.

Atributos

Atributo Valor Descrição
value machine-readable format Specifies the machine-readable translation of the content of the element
