Tag HTML <q>

Exemplo

Marque uma cotação curta:

<p>WWF's goal is to:
<q>Build a future where people live in harmony with nature.</q>
We hope they succeed.</p>

Definição e Uso

A tag <q> define uma pequena citação. Os navegadores normalmente inserem aspas entre aspas.

Atributos

Atributo Valor Descrição
cite URL Specifies the source URL of the quote

Exemplo

Use CSS para definir o estilo do elemento <param>:

<html>
<head>
<style>
q {
color: gray;
font-style: italic;
}
</style>
</head>
<body>

<p>WWF's goal is to:
<q>Build a future where people live in harmony with nature.</q>
We hope they succeed.</p>

</body>
</html>
