Tag HTML <tfoot>

Exemplo

Uma seção de rodapé em um documento:

<table>
<thead>
<tr>
<th>Month</th>
<th>Savings</th>
</tr>
</thead>
<tbody>
<tr>
<td>January</td>
<td>$100</td>
</tr>
<tr>
<td>February</td>
<td>$80</td>
</tr>
</tbody>
<tfoot>
<tr>
<td>Sum</td>
<td>$180</td>
</tr>
</tfoot>
</table>

Definição e Uso

A <tfoot> tag é usada para agrupar o conteúdo do rodapé em uma tabela HTML. O <tfoot> elemento é usado em conjunto com os elementos < thead> e <tbody> para especificar cada parte de uma tabela (rodapé, cabeçalho, corpo). Os navegadores podem usar esses elementos para permitir a rolagem do corpo da tabela independentemente do cabeçalho e rodapé. Além disso, ao imprimir uma tabela grande que abrange várias páginas, esses elementos podem permitir que o cabeçalho e o rodapé da tabela sejam impressos na parte superior e inferior de cada página.
