Tag HTML <tr>

Exemplo

Uma tabela HTML simples com três linhas; uma linha de cabeçalho e duas linhas de dados:

<table>
<tr>
<th>Month</th>
<th>Savings</th>
</tr>
<tr>
<td>January</td>
<td>$100</td>
</tr>
<tr>
<td>February</td>
<td>$80</td>
</tr>
</table>

Definição e Uso

A tag <tr> tag define uma linha em uma tabela HTML.

Um <tr> elemento contém um ou mais elementos <th> ou <td> .

Atributos

Atributo Valor Descrição
- - -

Exemplo

Use CSS para alinhar o conteúdo dentro de <tr>:

<table style="width:100%">
<tr>
<th>Month</th>
<th>Savings</th>
</tr>
<tr style="text-align:right">
<td>January</td>
<td>$100</td>
</tr>
</table>
