Tag HTML <head>

Exemplo

Um documento HTML simples, com uma tag <title> dentro da seção head:

<!DOCTYPE html>
<html lang="en">
<head>
<title>Title of the document</title>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

Definição e Uso

A tag <head> é um contêiner para metadados (dados sobre dados) e é colocado entre a tag <html> e a tag <body>. Metadados são dados sobre o documento HTML. Os metadados não são exibidos. Os metadados normalmente definem o título do documento, conjunto de caracteres, estilos, scripts e outras metainformações. Os seguintes elementos podem ir dentro do <head>elemento:

  • <title> (obrigatório em todos os documentos HTML)
  • <style>
  • <base>
  • <link>
  • <meta>
  • <script>
  • <noscript>
